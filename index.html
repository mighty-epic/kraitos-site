FAQ

What Kraitos is, plainly.

Is Kraitos a chatbot?

No. It is an operator layer for real computer tasks across desktop, browser, terminal, and remote surfaces.

Can it control my computer?

That is the direction: controlled tool access, visible state, and user-owned execution instead of a closed black box.

Why does the installer still say EmploAI?

The public brand is becoming Kraitos while the existing working desktop build still comes from the EmploAI codebase.

Where should backend traffic go?

Keep app/server traffic on the API host and use this domain for the public website as the migration settles.

Kraitos is the new public name for EmploAI.
