Installation Instructions

Deploying Kraitos

1. **Run the Installer**: Double-click `Kraitos-Desktop-Setup.msi` to launch the wizard.

2. **Security Prompts**: If prompted by Windows SmartScreen, click *More info* and choose *Run anyway*. As the branding name transitions, you may see older package hashes referencing the core EmploAI build signature.

3. **Backend Binding**: On launch, provide your local server API credentials to bind the desktop control loops.
